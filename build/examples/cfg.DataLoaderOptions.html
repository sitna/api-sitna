Ejemplo de uso de SITNA.cfg.DataLoaderOptions.

Control para añadir datos externos: servicios WMS y archivos de datos geográficos.

Se pueden añadir WMS la dirección del servicio o eligiendo un servicio de la lista de sugerencias.

Se pueden añadir datos de archivos buscándolos en el cuadro de diálogo o arrastrándolos y soltándolos dentro del área del mapa.
