Clase SITNA.cfg.SearchCadastralSourceExt

Se define en: sitna.js:1020
Sección relacionada: 2.3.1. Objeto de configuración de opciones del buscador
Sección principal: 2.3. Objeto de configuración global

Configuración del origen de datos auxiliar a la búsqueda de parcelas catastrales para la codificación de los nombres de municipio.

Esta clase no tiene constructor.

Puede consultar el ejemplo online.

Ejemplo:

`javascript { url: '//miServicioWFS/ogc/wfs', featurePrefix: 'IDENA',
featureType: ['Pol_ParcelaUrbana', 'Pol_ParcelaRustica', 'Pol_ParcelaMixta'], municipality: { featureType: 'Pol_Municipio', labelProperty: 'MUNICIPIO', idProperty: 'COD_MUNICIPIO'
}, queryProperties: { firstQueryWord: 'COD_MUNICIPIO', secondQueryWord: 'POLIGONO', thirdQueryWord: 'PARCELA' } }
